Arjan Audit

Arjan audit is a node module that helps you automate the auditing process of static sites during dev process using Google's Lighthouse 6

Get Started Docs

Test your static site's SEO score locally before deploying

How it Works

Arjan audit uses express to serve your site and google chrome-launcher to make a headless launch of google chrome and retrieve the audits from Lighthouse 6.

Audit Report

Arjan audit parses thes results returned from lighthouse 6 and generates a report. The report created by the arjan CLI formats the data nicely to the console and adds colors based on score and customized recommendations.

Get Started

  1. Install arjan globally (you only have to do this once)
  2. Go into your project's main directory and run an audit
