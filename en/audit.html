Arjan Audit

Helps you automate the auditing process of your static sites during development using Google's Lighthouse 6.

Get Started Docs

Test your static site's SEO score locally before deploying

How it Works

Arjan audit uses express to serve your site and Chrome-launcher to start Google Chrome in headless mode and retrieve the audits from Lighthouse 6.

Audit Report

Arjan audit parses thes results returned from lighthouse 6 and returns an organized subset with recommendations based on your score and threshold. The report created by the arjan CLI has a nice compact format and adds colors based on score.

Get Started

  1. If you haven't already, install Arjan globally
  2. Go into your project's directory and initialize Arjan (only do this once per project =)
  3. Run an audit
  4. For more info you can always run or checkout the docs
