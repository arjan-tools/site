Arjan Deploy

Arjan deploy is a tool that helps you deploy static sites to the AWS Cloud using Infrastructure as Code

Get Started

Leverage the power and cost savings of the AWS Cloud in your static sites

How it Works

Arjan deploy uses Infrastructure as Code templates to help you provision and deploy all the necessary resources for your site in the cloud.

Generally static sites in the cloud consist of an object storage solution (i.e. S3) with a public policy, a CDN, a DNS (from your domain name provider or your cloud provider) and optionally may contain a digital certificate.

Features

  • Connect to a global CDN network with low latency and high speeds. Learn more about Cloudfront
  • Add/verify TLS certificates with ease using AWS ACM
  • Add advanced routing options with route53
  • Cheap and reliable object storage AWS S3/li>
  • Manage your projectâ€™s infrastructure with a single JSON file AWS Cloudformation

Add a CDN, DNS, HTTPS to your site with only one command

For an easier development workflow we have defined some templates that include:

Dev

S3 root bucket with a public policy.

Test

S3 root bucket, www reroute bucket and a route53 hosted zone.

Production

CDN w/ Route53 DNS (https): Deploys s3 bucket, route53 DNS, a cloudfront distribution and creates TLS certificates in AWS ACM.

Get Started

  1. Install arjan globally (you only have to do this once)

  2. Go into your project's directory
  3. Initialize the project (you only have to do this once per project)
  4. Deploy your project
