What is Arjan Toolkit

Arjan is an open source project concerned with helping you have a good developer experience when working with static sites in plain HTML/CSS/JS. It helps you from the beginning, by setting up a development environment, up to deploying an optimized project to the cloud and updating itâ€™s content.

Arjan is easy to use and works with any static site. There is no need to adapt it to your project, learn a new framework or worry that something will break (Arjan has no dependencies). Just add it to a past project or start a new one in a second.

Dev Environment

Use the power of Webpackis in your static sites. Provides a local server with hot reloading + bundler. Supports JSX, ES2020, SASS, LESS, Coffeescript and others.

Plugins

Arjan has modular tools to help to you take your projects to production (Localize, Optimize, Audit, Deploy).

CLI

An intuitive and easy to use command line interface. Have full control of all the plugins and features in one place.

LOAD is an acronym for localize, optimize, audit, deploy. Four things that we repeatedly do to our static site projects to take them to production. The purpose of the load workflow is to allow you to continue doing what your doing (i.e. using HTML5, CSS3 and VanillaJS) and help you take your projects to production.
