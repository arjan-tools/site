Modular Tools for Static Websites

Optimize

A single tool for all of your static optimization needs

Some of the features include deleting unused css classes, Minifying and compressing static assets (css/js/html/images), automatically resizing/replacing images to their optimal sizes, converting/replacing images to webP, and more!

Translate

Localization, automatic translation, and collaboration of content in HTML websites

The flexible workflow allows you to export all of your website’s content to a spreadsheet and collaborate with your customers. Select from 54 available languages, localize, and translate your site’s content using neural machine translation.

Deploy

An easy and flexible way of deploying static websites in AWS using Cloudformation

It allows you to select multiple options for your website’s stack like adding a CDN or a digital certificate. Arjan also helps you import infrastructure of existing static websites in AWS into a cloudformation stack.

To get started install the Arjan CLI → npm i arjan-cli

All the tools can be used together with the arjan CLI, inside of a task runner like gulp or grunt, programmatically inside your application, or may also be deployed as a lambda function with an API and used as a github webhook. For additional information read the docs.
