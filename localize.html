Arjan Localize

A node module that automatically localizes and translates static sites. It may also work as a very basic content management system.

Get Started Docs

Try it Out

The Parser:

The parser in Arjan Localize get's all the existing IDs of html elements with text content and saves them as key value pairs in a JSON locale. If the element doesn't have an ID, Arjan automatically creates one.

The GUI:

A form with a dropzone made with Super Easy Forms which features a Node.js Lambda function as it's backend. The GUI is pretty limited as you can't update your translations, but it's good for a one time job especially if you don't like using the terminal.
